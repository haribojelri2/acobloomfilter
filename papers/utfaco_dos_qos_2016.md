# A Unified Ant Agent Framework for Solving DoS and QoS Problems (UTFACO)

## Citation
저자 미상 (ResearchGate ID: 309672057)  
Published: November 2016  
Publisher: IEEE  
ResearchGate: https://www.researchgate.net/publication/309672057

## 핵심 내용
- **UTFACO** (Unified Threat identification and Fault localization using ACO) 프레임워크 제안
- DoS 공격 탐지: IP traceback 기반 위협 소스 식별
- QoS 장애 위치 파악: 네트워크 내 결함 노드 추적
- **Bloom Filter 결합**: 탐색 공간 축소 및 패킷 필터링 속도 향상
- DARPA 네트워크 토폴로지 데이터셋으로 검증

## ACO + Bloom Filter 결합 방식
- Bloom Filter로 이미 탐색된 경로/노드를 빠르게 제외 → ACO 탐색 공간 축소
- 패킷 필터링 단계에서 Bloom Filter로 의심 패킷 사전 선별 후 ACO 정밀 탐색
